# Сравнение алгоритмов внешней сортировки на Java

Этот проект представляет собой графическое приложение на Java для сравнения четырёх алгоритмов внешней сортировки. Приложение позволяет сортировать большие текстовые файлы, визуализировать прогресс и сравнивать производительность алгоритмов в параллельном и последовательном режимах.

## Поддерживаемые алгоритмы

1. **One-Way Merge Sort** – классическое однонаправленное слияние
2. **K-Way Merge Sort** – многопутевое слияние с использованием приоритетной очереди
3. **Replacement Selection Sort** – алгоритм замены с созданием максимально длинных серий
4. **Bucket Sort** – блочная сортировка по длине слов

## Использование

### 1. Выбор файла
- Нажмите кнопку **"Выбрать файл"** для выбора текстового файла
- Поддерживаемые форматы: `.txt`, `.text`, `.log`, `.csv`, `.xml`, `.json`, `.md`, `.java`, `.cpp`, `.c`, `.py`, `.html`, `.css`, `.js`

### 2. Выбор режима сортировки
- **Параллельный режим**: все 4 алгоритма запускаются одновременно
- **Последовательный режим**: алгоритмы выполняются один за другим

### 3. Запуск сортировки
- Нажмите **"Начать сортировки"** для запуска процесса
- Прогресс отображается на индикаторах для каждого алгоритма
- Время выполнения каждого алгоритма отображается в реальном времени

### 4. Результаты
- Отсортированные файлы сохраняются в ту же директорию, что и исходный файл
- Имена выходных файлов: `[имя_файла]_sorted_[алгоритм].txt`
- По завершении отображается сводка по времени выполнения всех алгоритмов

## Алгоритмы сортировки

### One-Way Merge Sort
- Разделяет файл на отсортированные чанки
- Последовательно сливает пары чанков до получения единого файла
- Прост в реализации, но не самый эффективный для больших файлов

### K-Way Merge Sort
- Использует приоритетную очередь для одновременного слияния K чанков
- Эффективен для большого количества чанков
- Используется другими алгоритмами для финального слияния

### Replacement Selection Sort
- Создает максимально длинные серии (runs) из данных
- Использует два буфера для текущей и следующей серии
- Генерирует меньше серий, чем другие алгоритмы

### Bucket Sort
- Распределяет слова по корзинам на основе их длины
- Сортирует каждую корзину отдельно
- Объединяет отсортированные корзины в один файл
